# Neemee

**A lightweight notes app with portable markdown and full AI integration**

> **Note:** This repository is for community feedback, bug reports, and feature requests only. The Neemee source code is proprietary and maintained in a private repository.

---

## 🌟 What is Neemee?

Neemee is a lightweight alternative to Google Docs and Apple Notes with full bidirectional AI integration. Create, edit, and manage notes from our app or directly from Claude Code, ChatGPT, Codex, and any other MCP-enabled AI tool.

**Website:** [neemee.app](https://neemee.app)

### The Problem

Most note-taking apps trap your data in proprietary formats. Google Docs stores everything in their cloud format. Apple Notes uses its own database structure. When you want to export your notes, you get limited options—often losing formatting, metadata, and structure in the process. Using your notes with different tools or AI assistants means you're stuck copying and pasting or working around export limitations. Your notes should work everywhere, not just in one company's ecosystem.

These apps have also become bloated with features most people don't need. You just want a simple, fast way to take notes and access them from anywhere - including from AI tools you use every day. You need a lightweight alternative built on open standards that gives you complete data portability and works seamlessly with modern AI workflows.

### Our Solution

Through the Model Context Protocol (MCP), your AI assistants can read your notes for context, create new notes from conversations, update existing content, and organize everything—all while you maintain complete control over when and how your notes are modified.

Your notes are stored in **portable markdown format** with YAML frontmatter metadata, not trapped in proprietary systems. Export anytime, use with any tool, and keep your data truly yours.

## ✨ Key Features

- **🤖 Full AI Integration** - Seamless bidirectional integration with Claude Code, ChatGPT, and any MCP-enabled AI tool
- **📝 Rich Markdown Editor** - Powerful editor with live preview, syntax highlighting, and formatting tools
- **🔖 Single-Click Web Clipper** - Capture web content instantly with our bookmarklet
- **📦 Portable Format** - All notes stored as standard markdown with YAML frontmatter—no vendor lock-in
- **🔍 Powerful Search** - Full-text search with filtering by tags, domains, notebooks, and dates
- **🕐 Version History** - Automatic backups of every save (up to 50 versions per note)
- **🏷️ Flexible Metadata** - Custom frontmatter fields for rich structured data
